# Meteora

A modern and responsive weather application built with React and TypeScript. The app provides real-time weather information and forecasts using the OpenWeather API through a clean and user-friendly dashboard.

## Technologies Used

* React
* TypeScript
* Tailwind CSS
* TanStack Query (React Query)
* OpenWeather API
* React Router

## Features

* Search for cities worldwide
* Real-time weather information
* Weather forecast dashboard
* Favorite cities management
* Search history
* Responsive design for desktop and mobile devices

## Running the Project Locally

If you would like to download and run this project on your machine, please follow these steps:

### 1. Get an API Key

Create a free API key from OpenWeather:

https://openweathermap.org/api

### 2. Create an Environment File

Create a `.env` file inside the `app` folder.

### 3. Add Your API Key

```env
VITE_OPENWEATHER_API_KEY=your_api_key_here
```

Replace `your_api_key_here` with your OpenWeather API key.

Do not use quotes (`""`) or add extra spaces.

### 4. Start the Development Server

cd app
npm run dev
```

The application should now be running locally.
